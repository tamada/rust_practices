# lgrep

Lesser `grep`

```
lgrep 0.1.0
Haruaki Tamada <tamada@users.noreply.github.com>
finds and prints the line which matches the given pattern.

USAGE:
    lgrep [FLAGS] <PATTERN> [FILEs]...

FLAGS:
    -f, --fixed          search PATTERN as a fixed string.
    -h, --help           Prints help information
    -i, --ignore-case    case sensitive matching
    -V, --version        Prints version information

ARGS:
    <PATTERN>     specifies the pattern for matching.
    <FILEs>...
```
