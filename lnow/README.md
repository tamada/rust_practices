# now

```shell
now 0.1.0
Haruaki Tamada <tamada@users.noreply.github.com>
display the date and time.

USAGE:
    now [FLAGS] [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -u, --utc        Display the date and time in UTC (coordinate universal) time.
    -V, --version    Prints version information

OPTIONS:
    -f, --format <FORMAT>        Display the date and time in specified format. Available values are: ISO, Long, Short,
                                 and Minimal
    -R, --relative <TIME>        Display the date and time in the relative time of the local time, such as -9:00 (shows
                                 UTC if the local time is in Asia/Tokyo)
    -r, --relative-utc <TIME>    Display the date and time in the relative time of UTC, such as +9:00 (means Asia/Tokyo)
    -t, --timezone <TZ>          Display the date and time in the specified timezone, such as Asia/Tokyo.
```
