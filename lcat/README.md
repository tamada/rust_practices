# Lcat

Lesser `cat` command.

```sh
$ lcat -h
lcat 0.1.0
Haruaki Tamada <tamada@users.noreply.github.com>
Concatnate the given files.

USAGE:
    vat [FLAGS] [FILEs]...

FLAGS:
    -h, --help                 Prints help information
    -b, --numbers-non-blank    Prints the non-blank line with line numbers, start at 1.
    -n, --numbers              Prints the line with line numbers, start at 1.
    -s, --squeeze              Squeezes multiple adjacent empty lines, causing the output to be single .
    -V, --version              Prints version information

ARGS:
    <FILEs>...    Files for printing the contents.
```
