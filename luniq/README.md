# uniq

Delete duplicated lines.

```
uniq 0.1.0
Haruaki TAMADA
Delete duplicated lines.

USAGE:
    uniq [FLAGS] [ARGS]

FLAGS:
    -a, --adjacent        Deletes only adjacent duplicated lines.
    -d, --delete-lines    Only prints deleted lines.
    -h, --help            Prints help information
    -i, --ignore-case     Case insensitive.
    -V, --version         Prints version information

ARGS:
    <INPUT>     gives file name of input.  If argument is single dash ('-') or absent, the program read strings from
                stdin.
    <OUTPUT>    represents the destination.
```
